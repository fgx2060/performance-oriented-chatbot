# performance-oriented-chatbot
人工智慧實務實作 - 以效能為導向的客製化聊天機器人

透過 LangChain + LoRA/QLoRA 微調，建立高效能的領域客製化聊天機器人。

- 支援 RAG + LangChain
- LoRA / QLoRA 模型微調
- 高效能推論與回覆

## 🛠️ 技術架構
- **模型**：LoRA / QLoRA
- **框架**：LangChain
- **部署**：本地 / 雲端
